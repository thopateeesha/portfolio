Check out the live version of the portfolio: [https://thopateeesha.github.io/portfolio/)

## 🛠️ Built With

- **Frontend**: HTML5,CSS3,JavaSript
- **Icons**: custom

## 📁 Features

- **Interactive UI**: Smooth animations and transitions for better UX.
